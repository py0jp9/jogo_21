Blackjack em Python

Jogo de Blackjack (21) no terminal feito em Python puro, sem bibliotecas externas.

Como rodar

bashpython blackjack.py

Como jogar

Você recebe duas cartas e tenta chegar o mais perto de 21 sem passar. A cada rodada escolhe:


P → Pedir mais uma carta
S → Parar e deixar o Dealer jogar


Quem chegar mais perto de 21 sem passar vence.

Observações


Ás vale 11, mas vira 1 automaticamente se você passar de 21
J, Q e K valem 10
Projeto feito para praticar lógica em Python
